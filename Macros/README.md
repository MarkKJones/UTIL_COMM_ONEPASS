get_elastic.C : Calculates kinematics for a A(e,e') process to be elastic
                given the target, beam energy and electron angle
		Usage: >> root -l
                       >> .L get_elastic.C
                       >> get_elastic("", ebeam, eAngle)
		       ex. get_elastic("h", 10.59, 25.4)
